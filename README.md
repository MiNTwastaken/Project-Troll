# Projekta nosaukums
Project Troll
## Apraksts
	Palaižot failu,
	parāda vienu bildi
	Bilde tiek izvēlēta pēc nejaušības.
	Pēc tā parāda citu bildi (hehe)
## Galvenās funkcijas
	- Palaist
	- Atvērt pārlūkprogrammu
	- Atvērt saiti
  	- Nolādēt failu
  	- Atvērt failu kā video
## Izmantotās tehnoloģijas
	- C++
	- SFML library
